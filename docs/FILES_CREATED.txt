                              GETTING STARTED
═══════════════════════════════════════════════════════════════════════════════

1. Start Development Server
   $ npm run dev
   Open http://localhost:3000

2. Replace Placeholder Image
   • Save your headshot as public/hero.jpg
   • Recommended: 800x1000px JPG format
   • Refresh browser to see changes

3. Customize Text
   • Edit src/components/Hero.tsx (lines 81-115)
   • Change heading, subtitle, status text

4. Adjust Colors
   • Edit tailwind.config.ts
   • Modify cybersage color palette

5. Build for Production
   $ npm run build
   $ npm start

6. Deploy
   • Vercel: npx vercel
   • Netlify: Connect git repo
   • Other: npm run build, then deploy .next folder
